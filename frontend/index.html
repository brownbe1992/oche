Lifetime stats

Lifetime totals across every leg, set and game you've played on this device.

how each stat is counted
3-dart average — lifetime: all points scored ÷ all turns, across every leg, set and game. (During play, the scoreboard shows your current leg and current game averages instead.)
Ton+ finishes — winning turns where you scored 100 or more on that final visit.
Treble-less turns — turns where none of the darts landed in a treble bed.
A bust turn still counts as a turn (it scores 0) and still counts its darts toward treble-less totals.
